# Seven Layers

The AI Architecture Stack has seven layers.

Layer 1 is Physical Foundation: energy, cooling, facilities, and hardware environment.

Layer 2 is Chips and Accelerators: GPUs, TPUs, NPUs, CPUs, memory, and acceleration hardware.

Layer 3 is Compute Infrastructure: cloud, on-premise, local, and edge runtime environments.

Layer 4 is Data and Knowledge: data sources, pipelines, retrieval, grounding, and context.

Layer 5 is Model and Learning: model selection, algorithm development, training, tuning, and evaluation.

Layer 6 is Orchestration and Agents: workflow logic, tool use, task coordination, and multi-step execution.

Layer 7 is Application and Experience: user interface, integration, feedback, and practical value.

Use the layers to locate your work. Algorithm development belongs mainly to Layer 5. Retrieval and knowledge grounding belong mainly to Layer 4. Agent workflows belong mainly to Layer 6. Deployment choices belong mainly to Layer 3. Hardware and edge optimisation involve Layers 1, 2, and 3.
