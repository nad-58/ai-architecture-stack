# Layers Overview

The framework uses seven levels:

1. Physical Foundation
2. Hardware Accelerators
3. Compute Infrastructure
4. Data and Knowledge
5. Model and Learning
6. Orchestration
7. Application and Experience

Use this guide to identify where a task belongs. Algorithm work is mainly in the model layer. Data preparation and retrieval are in the data layer. Deployment decisions are in the compute infrastructure layer. User interfaces and adoption are in the application layer.
